# 비상탈출 시뮬레이터 README

## 프로젝트 설명

이 프로그램은 3층 건물 구조를 배열로 저장하고, 사용자가 시작 위치를 선택하면 BFS 알고리즘을 이용해 가장 가까운 출구까지의 경로를 찾는 비상탈출 시뮬레이터이다.

지도는 3차원 배열로 구성되어 있으며, 각 값은 빈 공간, 벽, 계단, 출구를 의미한다. 사용자는 방향키와 키보드 입력을 통해 시작 위치를 정하고, 프로그램은 해당 위치에서 출구까지 이동 경로를 계산해 화면에 출력한다.

## 주요 기능

* 3층 건물 지도 출력
* 사용자 시작 위치 선택
* BFS를 이용한 최단 탈출 경로 탐색
* 계단을 통한 층 이동 처리
* 탈출 경로를 화면에 순서대로 표시

## 함수 설명

### `main()`

프로그램의 시작 함수이다.

처음 화면을 출력한 뒤 사용자의 시작 위치를 입력받고, 이후 BFS 함수를 실행하여 탈출 경로를 탐색한다.

```c
int main()
```

실행 순서

1. `print_ui()`로 시작 화면 출력
2. `usr_input()`으로 사용자 위치 입력
3. `BFS()`로 탈출 경로 탐색

---

### `check(int z, int y, int x)`

BFS 탐색 중 이동하려는 위치가 유효한지 확인하는 함수이다.

```c
int check(int z, int y, int x)
```

확인하는 내용

* 배열 범위를 벗어나는지 확인
* 음수 좌표인지 확인
* 해당 위치가 벽인지 확인

이동 가능한 위치이면 `1`을 반환하고, 
이동할 수 없는 위치이면 `0`을 반환한다.

---

### `BFS()`

사용자가 선택한 위치에서 출구까지의 최단 경로를 찾는 함수이다.

```c
int BFS()
```

큐를 이용해 현재 위치에서 이동할 수 있는 칸을 차례대로 탐색한다. 일반 위치에서는 상하좌우 네 방향으로 이동하고, 계단 위치에서는 위층과 아래층 이동까지 포함해 여섯 방향을 탐색한다.

출구를 찾으면 이동 기록을 `print_result()` 함수로 넘겨 경로를 출ㄹ력한다. 출구를 찾지 못하면 실패 메시지를 출력한다.

주요 역할

* 시작 위치를 큐에 저장
* 방문한 위치를 `visited` 배열에 기록
* 이동 가능한 위치를 계속 탐색
* 출구 발견 시 이동 경로 출력

---

### `print_ui()`

프로그램 시작 화면을 출력하는 함수이다.

```c
void print_ui()
```

시작 안내 화면을 출력하고, 사용자가 스페이스바를 누를 때까지 기다린다.  스페이스바가 입력되면 다음 단계로 넘어간다.

---

### `print_usr_input(int z, int y, int x)`

사용자가 시작 위치를 선택하는 화면을 출력하는 함수이다.

```c
void print_usr_input(int z, int y, int x)
```

현재 층과 지도를 출력하고, 사용자가 선택 중인 위치를 화면에 표시한다.

지도 출력 기준

* 빈 공간: 이동 가능
* 벽: 이동 불가
* 계단: 층 이동 가능
* 출구: 탈출 지점
* 현재 선택 위치: 사용자가 고른 시작 위치

---

### `usr_input()`

사용자의 키보드 입력을 받아 시작 위치를 정하는 함수이다.

```c
void usr_input()
```

방향키를 이용해 같은 층에서 위치를 이동할 수 있고, `z`, `x` 키를 이용해 층을 이동할 수 있다. 스페이스바를 누르면 현재 위치가 시작 위치로 확정된다.

입력 방식

* 방향키: 상하좌우 이동
* `z`: 아래층 이동
* `x`: 위층 이동
* 스페이스바: 위치 선택 완료

벽이나 출구 위치로는 이동할 수 없도록 조건을 검사한다.

---

### `print_result(int arr[], int arr_len)`

BFS로 찾은 탈출 경로를 화면에 출력하는 함수이다.

```c
void print_result(int arr[], int arr_len)
```

BFS 함수에서 저장한 이동 방향 배열을 따라가며 실제 이동 경로를 지도에 표시한다. 이동한 위치는 별도로 표시하고, 일정 시간마다 화면을 갱신하여 경로가 순서대로 나타나게 한다.

출구에 도착하면 탈출 성공 메시지를 출력한다.

---

## 사용한 주요 변수

### `map`

건물의 실제 구조를 저장하는 3차원 배열이다.

```c
int map[height][length][width]
```

값의 의미

* `0`: 빈 공간
* `1`: 벽
* `3`: 계단
* `4`: 출구

### `res_map`

결과 출력용 지도 배열이다. BFS 결과 경로를 표시할 때 사용한다.

### `visited`

BFS 탐색 중 이미 방문한 위치를 저장하는 배열이다. 같은 위치를 반복해서 탐색하지 않기 위해 사용한다.

### `queue`

BFS 탐색에 사용하는 큐이다. 현재 위치와 이동 기록을 함께 저장한다.

### `_data`

BFS 큐에 들어가는 정보를 저장하는 구조체이다.

```c
typedef struct _data {
    int z, y, x;
    int move[200];
    int move_cnt;
} _data;
```

저장하는 정보

* `z`, `y`, `x`: 현재 위치
* `move`: 지금까지 이동한 방향
* `move_cnt`: 이동 횟수
